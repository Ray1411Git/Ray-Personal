# Todo List - Versión práctica

**Descripción**
Interfaz semi estática de una aplicación de lista de tareas creada como práctica del bootcamp HackABoss. Incluye HTML semántico, CSS responsive y un script mínimo en JavaScript para interacción básica (añadir, marcar como completada, eliminar).

---

## Características
- Añadir tareas desde el formulario.
- Marcar tareas como completadas (tachado visual).
- Eliminar tareas individuales.
- Eliminar tareas completadas.
- Eliminar todas las tareas.
- Diseño responsive para móvil y escritorio.
- Accesibilidad básica con labels y atributos aria.

---

## Tecnologías
- **HTML5**
- **CSS3** (Flexbox, Grid, media queries)
- **JavaScript** (DOM puro, sin librerías)

---

## Cómo ejecutar
1. Clona o descarga el repositorio.
2. Abre la carpeta `todo-list`.
3. Abre `index.html` en tu navegador.
4. Interactúa con la app:
   - Escribe una tarea y pulsa **Añadir**.
   - Marca el checkbox para tachar la tarea.
   - Pulsa la cruz para eliminar una tarea.

---

## Notas de implementación
- El HTML incluye comentarios explicativos para cada bloque semántico.
- El CSS usa variables para mantener consistencia visual.
- El JS es intencionalmente simple y está comentado para facilitar la lectura.
- No se usa almacenamiento persistente; al recargar la página se pierden las tareas.

---
